------------------

#### 项目学习方向

* 硬件平台：`arduino` `stm32` `rasberry pi`
* 系统平台：`linux` `Windows`
* 软件平台：`qt5` `ucos` `opencv`
* 编程语言：不限，必须学习`c`、`c++`，推荐学习`python`、`java`
